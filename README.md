## What is MacroX?

**MacroX** is a Chrome extension that lets you record any sequence of browser interactions and replay them instantly — clicks, keystrokes, scrolls, form fills, and more. Think of it as a DVR for your browser.

No Selenium. No scripts. No setup. Just hit Record, do the thing, hit Stop, and press Play.

## How It Works

```
1. Open the MacroX sidebar (click the toolbar icon)
2. Press ⏺ Record — MacroX starts capturing your actions
3. Interact with any web page normally
4. Press ⏹ Stop (or F8) when done
5. Name your macro and assign an optional hotkey
6. Press ▶ Play any time to replay it perfectly
```

## Privacy

MacroX is **privacy-first**:

- All macro data is stored **locally** in `chrome.storage.local`
- **Nothing is ever transmitted** to any external server
- Password fields are **explicitly excluded** from recording
- Payment is handled by [ExtensionPay](https://extensionpay.com) / Stripe — we never see your card details

→ [Full Privacy Policy](https://sn3ll.github.io/macrox-chrome-extension/privacy.html)

## Permissions Explained

| Permission | Why |
|---|---|
| `storage` | Save macros and settings locally on your device |
| `activeTab` | Access the tab you're recording/playing on |
| `scripting` | Inject the recorder into tabs when you start recording |
| `sidePanel` | Show the MacroX UI in Chrome's side panel |
| `tabs` | Identify the active tab, resume after navigation |
| `<all_urls>` | Record and replay on any website you choose |
